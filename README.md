# Pyblog的开发版本

###开发环境

- 支持 Linux 系统，本程序默认为 Ubuntu 14.04 - 16.04 系统。
- 默认的python， 2.7.6 - 2.7.11+。
- Ubuntu 系统默认安装的nginx。
- 默认使用系统自带的 sqlite 数据库。
- IDE 工具是Pycharm 4.5
